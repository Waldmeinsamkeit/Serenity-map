# Industry Research Canvas Prompt Pack

## Theme Scan

```text
Use $industry-research-canvas to research [market] [theme].
Rank the value-chain layers first, identify scarce layers, map companies, grade evidence,
state risks and next checks, and return a Serenity canvas-ready structure.
```

## A-Share Scan

```text
用 $industry-research-canvas 研究当前 A 股 [行业/主题]。
先排产业链层级，再找最值得优先研究的公司。
请区分事实、推断和待验证项，并给出可以转成 Serenity canvas 的节点和边。
```

## Company Challenge

```text
用 $industry-research-canvas 挑战 [公司/代码] 的产业链地位。
它到底控制卡点、供应卡点、受益于需求，还是只是相邻题材？
请列出证据、缺口、风险和下一步验证。
```

## Canvas Patch

```text
Use $industry-research-canvas to convert this research into a Serenity AI Patch.
Use short card titles, detailed body fields, stable node ids, and semantic edge kinds.
```

## Data Source URLs

```text
Use $industry-research-canvas to choose provider URLs and public sources for [ticker/theme].
Explain which sources support filings, financials, valuation, capital flow, news, research reports,
and web evidence. Do not assume any local data-source repository exists.
```
